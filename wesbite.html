Web Project

Creating a team website is a requirement for third homework of Principles of Information Systems. To complete this homework, we should complete CSS and HTML courses attached by the instructor. We completed all the courses and started to create our page. For creating first skeleton of website, we used HTML, for decoration we used CSS. Finally, we utilized JavaScript for the animation part. About team name – Data Pirates, it is suggested by Sevda Aliyeva. Logo was mostly created by Rahimakhanim Ismayilzada.
